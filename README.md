# 社团管理系统简介 java706

## 项目概述

社团管理系统是一个基于Web的社团活动与信息管理平台，采用前后端分离的技术架构。该系统致力于为不同角色用户提供便捷的服务与高效的管理。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**：IDEA、JDK 1.8、Maven、Mysql 8.0
- **技术框架**：Java、SpringBoot、Mybatis、Vue

## 功能模块

### 用户角色

- 用户模块
  - 登录/注册
  - 用户信息管理

### 系统角色

- 超级管理员
  - 拥有所有社团管理的权限

- 社团管理员
  - 本社团信息管理
  - 社团活动管理
  - 入团申请处理
  - 费用管理
  - 社团活动通知

- 学生用户
  - 社团信息查看
  - 社团活动参与
  - 加入社团
  - 费用支付
